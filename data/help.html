System requirements

  1. OS: Linux with ping & grep tools
  2. Installed Mercurial with configured repos: [paths]
    default = http://hg.nixsolutions.com/%REPO%
    [auth]
    d.prefix = http://hg.nixsolutions.com/%REPO%
    d.username = %USER%
    d.password = %PATH%
  3. PHP 5.3.1+ with enabled: exec, passthru, shell_exec, curl (and basic ZFCore dependencies)
  4. Zend Framework 1.11.1 with Installed Zend Tool
